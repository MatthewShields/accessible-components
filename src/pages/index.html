What is Accessible Components?

A journey to build a library of accessible components and the approaches behind them. This will be built up over time in my journey to ensure that I am building a more accessible web.

This isn't an effort to create a component library, but more of a guide for when we are building components to ensure that the functionality is accessible for everyone.

What if you spot something wrong or can be improved?

As I have said, this site is to document my journey through learning how to make accessibility, and so there is a chance that there may be something I can improve on. I am doing a lot of research with each of the components I look at, but I would always be appreciative of learning more.

If you do see anything, you can reach me on Twitter, create an issue in the Git Repo or send me an email at accessiblecomponents@matthewshields.co.uk.
